# Direct Load (DL) vs No-DL Regression Investigation

Investigation of GEMM performance regressions when using
`--iree-llvmgpu-use-direct-load` on AMD gfx950.

Source data (47 regressed shapes): https://gist.github.com/Yu-Zhewen/fc858845ba77967877526edf728e5e1b

## Quick Start

```bash
# 1. Build IREE (see Build section below) and put it on PATH.

# 2. Sweep the full 47-case regression suite.
python3 run_ab.py --preset=full_regression \
    --b-flags="--iree-llvmgpu-use-direct-load" \
    --a-label=nodl --b-label=dl \
    --skip-n1 --max-output-bytes=4G \
    --out-csv=data/results.csv

# 3. Faster top-20 sweeps:
python3 run_ab.py --preset=non_tn_top20 --b-flags="--iree-llvmgpu-use-direct-load"
python3 run_ab.py --preset=tn_top20     --b-flags="--iree-llvmgpu-use-direct-load"

# 4. Investigate one shape in depth (IR + optional ISA):
python3 inspect_dl_codegen.py 1134 2048 150000 --transpose LHS_T --isa
```

### Generic IREE Utilities (in `~/scripts/iree/`)

These standalone scripts are shared with other reproducers and are
imported via a `sys.path` shim:

- `gen_matmul.py` — Generate MLIR for matmul / batch-matmul (any
  transpose, dtype, batch). Standalone CLI or importable.
- `iree_bench.py` — Compile MLIR to VMFB and benchmark with
  `iree-benchmark-module`. Standalone CLI or importable.
- `inspect_isa.py` — Parse stats (lines, VGPR/SGPR, opcode counts) from
  one or two `.rocmasm` files. Standalone CLI or importable; replaces
  the old `compare_isa.sh`.

### Related Tools (in `~/scripts/tools/`)

Useful for deeper profiling work that this reproducer does not cover:

- `prof.sh` — `rocprofv3` wrapper (supports `trace` and `att` modes).
- `att_analyze.py` — Parse ATT (Advanced Thread Trace) CSV results,
  report stall/latency/idle breakdown.
- `genRandInput.py` — Generate random binary inputs for IREE modules.
- `compare.py` — Compare binary outputs with configurable threshold.

## Build Instructions

```bash
cd /root/iree

cmake -B build/dbg -G Ninja \
  -DCMAKE_BUILD_TYPE=Debug \
  -DCMAKE_C_COMPILER=clang \
  -DCMAKE_CXX_COMPILER=clang++ \
  -DCMAKE_C_COMPILER_LAUNCHER=ccache \
  -DCMAKE_CXX_COMPILER_LAUNCHER=ccache \
  -DIREE_ENABLE_LLD=ON \
  -DIREE_HAL_DRIVER_HIP=ON \
  -DIREE_TARGET_BACKEND_ROCM=ON \
  -DIREE_TARGET_BACKEND_LLVM_CPU=ON

cmake --build build/dbg -- iree-compile iree-benchmark-module iree-run-module

export PATH=/root/iree/build/dbg/tools:$PATH
```

## Scripts

### `run_ab.py` — A/B Performance Comparison

Compiles each shape with two configurations (A=baseline, B=experiment),
skips benchmarking when both VMFBs are identical, and reports a
side-by-side ratio. Loads shapes from any preset CSV in `data/`
(auto-discovered) or from an arbitrary `--csv` path.

```bash
# Use a preset:
python3 run_ab.py --preset=non_tn_top20 \
    --b-flags="--iree-llvmgpu-use-direct-load"

# Custom labels:
python3 run_ab.py --preset=non_tn_top20 \
    --a-flags="" --b-flags="--iree-llvmgpu-use-direct-load" \
    --a-label=nodl --b-label=dl

# Reuse existing VMFBs (skip compilation):
python3 run_ab.py --preset=non_tn_top20 --skip-compile

# Run / skip specific shapes (1-based indices):
python3 run_ab.py --preset=non_tn_top20 --only=05,07,10
python3 run_ab.py --preset=non_tn_top20 --skip=09

# Filter degenerate shapes during full-suite sweeps:
python3 run_ab.py --preset=full_regression --skip-n1 --max-output-bytes=4G

# Save per-shape results to CSV:
python3 run_ab.py --preset=full_regression \
    --b-flags="--iree-llvmgpu-use-direct-load" \
    --out-csv=data/results.csv

# Load shapes from any CSV:
python3 run_ab.py --csv=my_shapes.csv
```

### `inspect_dl_codegen.py` — Single-Shape IR/ISA Inspector

For one shape, generates MLIR, compiles with and without
`--iree-llvmgpu-use-direct-load`, and:

- Dumps the IR before/after the prefetch pass to `_work/investigation/`.
- Reports whether software pipelining activated by counting
  `amdgpu.gather_to_lds` (DL mode) or `vector.transfer_read` (no-DL
  mode) ops before vs after the prefetch pass.
- With `--isa`, also extracts `.rocmasm` and prints a side-by-side
  register / opcode comparison via `~/scripts/iree/inspect_isa.py`.

```bash
python3 inspect_dl_codegen.py 1134 2048 150000 --transpose LHS_T
python3 inspect_dl_codegen.py 24576 2048 512 --transpose RHS_T --isa
python3 inspect_dl_codegen.py 64 64 64 --batch 8 --dtype bf16 --isa
```

## Data Files

All CSVs live in `data/` and follow the schema:

```
name,M,N,K,transpose,batch,dtype_lhs,dtype_rhs
```

`name` is used for output filenames; the other columns drive
`gen_matmul`. To add a new preset, drop a CSV in `data/`; it will be
auto-discovered by `run_ab.py --preset`.

- `data/full_regression.csv` — All 47 regressed shapes.
- `data/non_tn_top20.csv` — Top 20 non-TN (NN + RHS_T) shapes.
- `data/tn_top20.csv` — Top 20 TN (LHS_T) shapes.
- `data/results.csv` — Generated by `run_ab.py --out-csv`.
